Version: Következő

Telepítés

Önálló parancsfájl használata#

curl -f https://get.pnpm.io/v6.js | node - add --global pnpm

Windows rendszeren (PowerShell):

(Invoke-WebRequest 'https://get.pnpm.io/v6.js' -UseBasicParsing).Content | node - add --global pnpm

Az önálló szkript alá van írva. Így ellenőrizheti.

npm használatával#

npm install -g pnpm

Via npx resolution#

npx pnpm add -g pnpm

Frissítés#

pnpm telepítése után nincs szükség más csomagkezelők használatára a frissítéséhez. A pnpm-et saját maga is frissítheti, így:

pnpm add -g pnpm
tip

Szeretné használni pnpm-et CI szervereken? Lásd: Folyamatos integráció.
